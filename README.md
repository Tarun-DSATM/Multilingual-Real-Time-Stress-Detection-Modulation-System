# Multilingual-Real-Time-Stress-Detection-Modulation-System
– Built a real-time speech-based stress detection system supporting Kannada, Telugu, Tamil, Malayalam and additional
Indian languages using spontaneous speech from 125 native speakers.
– Developed a BiLSTM-Attention model fine-tuned on IndicWav2Vec (40+ Indian languages) and IndicVoices corpus,
achieving 75% accuracy and 92.3% F1-score.
– Designed adaptive interventions with multilingual TTS, guided breathing, calming music, and a closed-loop feedback
system with less than 300ms latency.
– Tech Stack: Python, PyTorch, Flask, HTML, CSS, JavaScript, React, TypeScript, Supabase (DB & Auth), Node.js.
